# example_node_typescript_postrgresql

Archetype for NODE Typescript and PosgreSQL with an user CRUD example API

## Installation

Run
```bash
npm -i
```

## Usage

create .env in root path with the following env variables, you must complete env values:
```bash
ENVIRONMENT=dev
API_PATH=
API_HOST=
API_PORT=
DB_NAME=
DB_HOST=
DB_PORT=
DB_PSW=
DB_USER=
DB_MAX_CLIENTS=
DB_IDLE_TIMEOUT_MS=
```

Run
```bash
npm run start
```
